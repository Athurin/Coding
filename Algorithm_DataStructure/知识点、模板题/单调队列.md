
找最小值，队列的内部是单调递增的
```cpp
a[]//原始数组
q[]//存放队列,这里一般存放 a[] 元素的下标 
//单调递增的队列，队头存放最小值
int hh = 0, tt = -1;
for(int i=0; i<n; i++)
{
	//判断队头是否处在窗口内部
	// k 是窗口的长度
	if(hh <= tt && i-k+1 > q[hh]) hh++;
	
	//时刻保持单调性
	while(hh <= tt && a[q[tt]] >= a[i]) tt--;
	//队尾更新
	q[++tt] = i;
	
	//这里一般会加入单调队列想要实现的操作

}
```
