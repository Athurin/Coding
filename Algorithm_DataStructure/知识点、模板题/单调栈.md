
只有一端可操作的称为栈，栈内元素全部单调排列的称为单调栈

单调递减的，找最小值
```cpp
a[]//原始数组 
stk[N], tt;//单调栈，存放 a[] 的元素下标
// tt 为栈顶，tt == 0 表示栈为空
//所以有效的下标从 1 开始
for(int i=1; i<=n; i++)
{
	while(tt && b[i] >= b[stk[tt]])
	{
		tt--;
		//加上其他操作
	}
	//栈顶更新
	stk[++tt] = i;
}
```

单调递增的，找最大值
```cpp
a[]//原始数组 
stk[N], tt;//单调栈，存放 a[] 的元素下标
// tt 为栈顶，tt == 0 表示栈为空
//所以有效的下标从 1 开始
for(int i=1; i<=n; i++)
{
	while(tt && b[i] <= b[stk[tt]])
	{
		tt--;
		//加上其他操作
	}
	//栈顶更新
	stk[++tt] = i;
}
```

应用情景，找到某个数往左或者往右第一个把他大或者比他小的数的下标。
